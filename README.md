# SBMM Boring Explorer

Interactive web map of all previous geotechnical exploration borings at the **Sulphur Bank Mercury Mine (SBMM) Superfund Site**, Clearlake Oaks, Lake County, California.

## Features

- **Interactive map** with 101 previous boring locations plotted on ESRI satellite imagery
- **Color-coded markers** by Figure 3-1 subproject area (South Cell, North Cell, Northwest Cell, Disturbed Rock & Northern WRD)
- **Filter by** subproject area, investigation year, drilling company, or well type
- **Search** by boring ID
- **Detail panel** with all boring data: coordinates, depths, groundwater, screen interval, company, condition
- **Responsive design** — works on desktop and mobile

## Data Sources

Compiled from 5 source documents spanning 1988–2024:
- Summary of Existing Explorations spreadsheet (97 records)
- EA Engineering Table A-1 monitoring well summary (60 wells)
- Historical Boring Info PDF (287 pages, 33 detailed logs)
- SBMM.kmz spatial data (255 placemarks)
- SBMM Geotechnical Report in progress
